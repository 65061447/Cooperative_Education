# Employee Management System (React + Express)

A simple employee management system built with **React (Frontend)** and **Express + MySQL (Backend)**.  
Designed for internal use with authentication, filtering, and CRUD operations.

---

## 🚀 Features

- 🔐 Authentication (JWT-based)
- 👥 Employee CRUD (Add / Edit / Delete)
- 🔍 Search & Filter (Name, Position, Division, Status, etc.)
- 📊 Sorting (Entry Date, Level, Gen, Position No.)
- 🧠 Client-side filtering for fast performance
- 🎨 Clean UI with Tailwind + shadcn components

---

## ⚙️ Tech Stack

### Frontend
- React (Vite)
- TypeScript
- Tailwind CSS
- shadcn/ui
- lucide-react icons

### Backend
- Node.js
- Express
- MySQL (mysql2)
- JWT Authentication
- Helmet + CORS
